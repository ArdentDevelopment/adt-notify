# adt-notify

Simple notification system for FiveM.

![preview](https://www.youtube.com/watch?v=r5dcQPOyKKQ)

## Installation

1. Drop `adt-notify` into your resources folder
2. Add `ensure adt-notify` to `server.cfg`
3. Restart Your server
